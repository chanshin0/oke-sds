🏪 Marketplace — 카탈로그 + 진입점

App Store 비유. plugin 들의 진열대 + 유일한 install 통로

Marketplace 가 하는 일
• .claude-plugin/marketplace.json 단 한 파일
• plugin 들의 이름 + source 만 적힌 카탈로그
• 실제 plugin 코드는 안 들어있음 — 포인터만
vs
Plugin 이 하는 일
• 실제 도구 코드 (skills/, commands/ 등)
• .claude-plugin/plugin.json 메타
• marketplace 통해야만 install 됨
사용자 입장
/plugin marketplace add <repo>
→ 카탈로그 등록 (plugin 0개 깔림)
+
그 다음
/plugin install <name>@<market>
→ 골라서 개별 install
Marketplace 자체는 코드 0줄. plugin 들 어디서 끌어올지 가리키는 포인터 모음. plugin 과 같은 repo 에 있어도 되고, 다른 repo 가리켜도 됨.
