📦 Plugin — 도구 묶음

하나의 plugin = 여러 종류의 도구가 함께 들어있는 패키지

🧠
skills/
자동 트리거 도구. 각 폴더가 SKILL.md 보유
⚡
commands/
명시적 슬래시 호출 (/<plugin>:<cmd>)
🤖
agents/
전문 작업용 sub-agent — 격리된 컨텍스트에서 실행
🪝
hooks/
PreToolUse / PostToolUse / Stop 등 이벤트 훅
🔌
.mcp.json
MCP 서버 (외부 도구 통합)
📋
.claude-plugin/
plugin.json
메타 (이름·설명·author·license). 이게 있어야 plugin
필요한 것만 골라 담을 수 있음. skill 만 있는 plugin, command 만 있는 plugin 모두 가능.
