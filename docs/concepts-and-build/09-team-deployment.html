팀 배포 — 4 단계

팀원이 클릭 몇 번으로 install 끝나는 흐름

1
팀 저장소 .claude/settings.json 에 marketplace 사전 등록
"extraKnownMarketplaces": { "oke-sds": { "source": { "github", "chanshin0/oke-sds" }, "autoUpdate": true } }
"enabledPlugins" 까지 미리 채워두면 더 매끄러움
2
팀원이 dev 풀 + Claude Code 시작
git pull → claude
Claude Code 가 settings.json 의 extraKnownMarketplaces 자동 감지
3
트러스트 + install 프롬프트 체인
Trust this folder? → Yes → Install marketplace? → Yes → Install plugin? → Yes
public marketplace 라 인증 불필요. private 면 GITHUB_TOKEN 필요
4
반영 + 이후 자동 갱신
/reload-plugins
marketplace 의 auto-update 토글 ON 해두면 이후 시작할 때마다 최신 자동 반영. version 필드 없음 → 매 commit = 새 version 으로 인식
